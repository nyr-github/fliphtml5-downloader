# Fliphtml5 Downloader - 轻松将 FlipHTML5 翻页书转换为 PDF

## 你好！👋

我是一名独立开发者，平时经常需要在 FlipHTML5 上查阅一些技术文档和电子杂志。但有个问题一直困扰我：这些在线翻页书虽然浏览体验不错，但想要离线保存、做笔记或者打印的时候就很麻烦了。

每次我只能一页一页地截图，或者用浏览器的打印功能手动保存，效率很低。后来发现 FlipHTML5 的资源其实是加密的，直接下载行不通。

于是我就想，能不能做一个工具来自动完成这个过程？花了一些时间研究他们的加密机制，最终做出了这个工具。

## 我们遇到的问题

如果你也用过 FlipHTML5，可能遇到过这些情况：

- **无法离线保存**：优秀的翻页书想收藏起来慢慢看，但只能在线浏览，断网就没法访问
- **做笔记困难**：想在文档上标注重点、写批注，但在线阅读器功能有限
- **打印不友好**：想打印部分内容，但每页都要手动操作，几十页的书会非常耗时
- **资源加密**：FlipHTML5 对页面资源进行了 WASM 加密，无法直接获取原始图片
- **批量处理麻烦**：有多个文档需要转换时，重复操作很繁琐

## 我做了什么

我做了一个基于 Next.js 的在线工具，可以自动解析 FlipHTML5 链接并转换成 PDF。

### 技术实现思路

这个工具的核心是解密 FlipHTML5 的加密配置：

1. **加载官方解密脚本**：使用 FlipHTML5 官方的 jQuery 和 deString.js 解密引擎
2. **WASM 内存操作**：通过 `allocateUTF8` 将加密字符串写入 WASM 内存
3. **调用解密函数**：执行 `Module._DeString` 进行字节反转解密
4. **提取页面数据**：解析解密后的 JSON，获取所有页面的图片 URL 和元数据
5. **生成 PDF**：使用 jsPDF 将页面图片按顺序整合成 PDF 文档

### 主要功能

- **URL 自动解析**：粘贴 FlipHTML5 链接，自动识别和提取书籍信息
- **PDF 生成**：将翻页书的每一页转换为高质量 PDF，保持原始排版
- **批量下载**：支持同时处理多个转换任务，不用等待
- **ZIP 资源兼容**：支持处理 ZIP 压缩格式的页面资源
- **社区发现**：可以浏览其他用户下载的热门出版物，发现好内容
