### How It Works (The Simple Version)

1. You paste a FlipHTML5 URL
2. We convert it to PDF
3. You download and keep it

That's it. Three steps. Ten seconds. Permanent access.

### How It Works (The Technical Version)

Our tool uses sophisticated technology:

**Phase 1: Configuration Retrieval**
- Fetch the FlipHTML5 publication
- Extract encrypted configuration data
- Identify all page resources

**Phase 2: WebAssembly Decryption**
- Load official FlipHTML5 decryption scripts
- Initialize WASM decryption module
- Reverse byte-inversion encryption
- Extract page URLs and metadata

**Phase 3: PDF Generation**
- Download high-resolution page images
- Optimize and format pages
- Assemble into structured PDF
- Deliver to user's browser

**All processing happens in your browser:**
- We never see your downloads
- No server-side storage
- Complete privacy
- Zero data collection
